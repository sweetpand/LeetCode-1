## 581. Shortest Unsorted Continuous Subarray [$\star$](https://leetcode.com/problems/shortest-unsorted-continuous-subarray) :thumbsup:

- [x] Time: $O(n)$
- [x] Space: $O(1)$

```python
class Solution:
    def findUnsortedSubarray(self, nums: List[int]) -> int:
        mini = float('inf')
        maxi = float('-inf')
        flag = False

        for i in range(1, len(nums)):
            if nums[i] < nums[i - 1]:
                flag = True
            if flag:
                mini = min(mini, nums[i])

        flag = False

        for i in range(len(nums) - 2, -1, -1):
            if nums[i] > nums[i + 1]:
                flag = True
            if flag:
                maxi = max(maxi, nums[i])

        for l in range(len(nums)):
            if nums[l] > mini:
                break

        for r in range(len(nums) - 1, -1, -1):
            if nums[r] < maxi:
                break

        return 0 if l >= r else r - l + 1
```

## 583. Delete Operation for Two Strings [$\star\star$](https://leetcode.com/problems/delete-operation-for-two-strings) :thumbsup:

```python
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m = len(word1)
        n = len(word2)

        dp = [0] * (n + 1)

        for j in range(n + 1):
            dp[j] = j

        for i in range(1, m + 1):
            newDp = [i] + [0] * n
            for j in range(1, n + 1):
                if word1[i - 1] == word2[j - 1]:
                    newDp[j] = dp[j - 1]
                else:
                    newDp[j] = min(newDp[j - 1], dp[j]) + 1
            dp = newDp

        return dp[n]
```
